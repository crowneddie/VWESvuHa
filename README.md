# 项目简介

## 核心功能
- 支持多种输入源检测：图片（单张或批量）、视频文件、本地摄像头实时检测
- 可自定义检测参数：置信度阈值、IOU阈值调整
- 多种显示选项：可选择显示标签和原图
- 模型选择功能：支持切换不同检测模型
- 实时信息展示：界面显示目标位置、目标总数、置信度等
- 批量处理能力：支持批量检测并查看所有结果
- 结果保存功能：可保存检测结果

## 适用场景
- 安防监控：实时监测攀爬行为，提高安全性
- 视频分析：对已有视频文件进行攀爬行为检测
- 批量图片处理：对大量图片进行自动化检测分析
- 实时监控：通过摄像头进行实时攀爬行为识别
- 研究教学：可用于计算机视觉学习与研究

## 技术特点
- 基于YOLOv8目标检测框架
- 使用Python3.9开发，依赖opencv-python、PyQt5、ultralytics、torch1.9等库
- 提供完整UI界面，操作简便
- 包含10000张标注数据集，分为'no_climb'和'climb'两类
- 预训练模型可直接使用，无需重新训练
